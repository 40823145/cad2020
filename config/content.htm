Develop

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.

可攜系統與網頁製作:

1. 下載fall2020_cp.7z壓縮檔，解壓縮

從Git網站下載PortableGit，解壓縮放入data中PortableGit資料夾

2. 修改start中git路徑

設定gitconfig

3. 儲存後重新啟動

4. git clone自己倉儲的網址

5. 進入github創建cad2020倉儲

6. cad2020資料夾創建cmsimde資料夾(黑窗)

git submodule add https://github.com/mdecourse/cmsimde.git

7. 複製cmsimde中up_dir資料夾內容至cad2020，不取代重複檔案

8. 下在網站需要的模組(黑窗)

pip install flask flask_cors lxml bs4 markdown pelican leo 

9. 進入cmimde，wsgi.py啟動動態網頁localhost9443

Raspberry Pi NAS

材料:Raspberry pi、SD card、硬碟

Step1:在Raspberrypi Official website下載Imager燒錄或下載ISO檔自行燒錄，燒錄完成後新增ssh文件(無副檔名)，複製進SD card的boot中，退出SD card插入樹莓派接上網路線開機

Step2:在路由器中搜尋樹莓派ip(若在樹梅派中找ip輸入ifconfig)，使用putty ssh輸入樹莓派ip連接

Step3:登入樹莓派(預設帳號:pi 密碼:raspberry)，登入後update upgrade，升級完成安裝NAS系統openmediavault，安裝完成ip可能會改變，需再查詢一次ip

sudo apt update
sudo apt upgrade
sudo wget -O - https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash

Step4:在電腦的瀏覽器輸入查詢到的ip(若無法進入，試試其他瀏覽器)，登入openmediavault(預設帳號:admin 密碼:openmediavault)，硬碟樹莓派，完成
